Here are the keys for sonic starships:

  • [ `, 1-9 ]: Set pixel resolution
  • [ - ] and [ + ]: Raise or lower gameplay speed
  • [ 0 ]: reset gameplay speed
  • [ SPACE ]: Toggle continuous star and 24 frets
  • [ W ]: Toggle volume speed
  • [ H ]: Full screen
  • [ V ]: Verbose
  • [ I ]: Increase zoom In
  • [ O ]: Increase zoom Out
  • [ P ]: Set zoom threshold to average for next second

    For Manny and MannyWomanny:

  • [ T ]: Touch to Move
  • [ Q ]: Cycle tree of Life colorings
  • [ N ]: Toggle popping clovers!
  • [ L ]: Set zoom to freerange
  • [ C ]: Keep focus on Center
  • [ J ]: Carousel
  • [ K ]: Multi-Carousel
  • [ ? ]: Spiration
  • [ ; ]: Reverse Rainbow
  • [ S ]: Engage morphing! This also works with the Mandelbrot.
  • [ Y ] and [ U ]: Remove or add a petal
