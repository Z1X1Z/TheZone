Zone:X is a game\visualizer which turns your voice into color and motion. I had to do some chinanegery to get the pitch detection working on this edition. It doesn't make sense how it works, but it seems to be about on point. Again, accuracy not guaranteed though it should work pretty well. If you are concerned about the accuracy you can compare it to the web edition which uses a correctly implemented YIN (from Yin Yang)detection method. Bon appetit!
